### Built an Artifact

For Java Maven and Gradle are used for compiling, compressing code, install dependencies, 

To build with gralde

``` ./gradlew build ``` 

build.gradle is the dependencies file

To build Maven project

``` mvn install ```

pom.xml is the dependencies file
